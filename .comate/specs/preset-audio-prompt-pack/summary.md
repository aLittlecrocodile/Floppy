# 预置缓存音频生成文案任务总结

## 完成内容

本次完成了预置缓存音频生成前的文案与提示词准备工作，没有修改业务代码。

已完成交付物：

- `.comate/specs/preset-audio-prompt-pack/tasks.md`

## 主要产出

### 1. 试生成范围

文档覆盖四类内容：

1. 白噪音 / 环境音
2. 轻音乐
3. 冥想
4. 睡前故事及助眠播客

细分品类包括：

- 环境音：雨声、海浪、溪流/瀑布、火焰、森林、空调/风扇
- 轻音乐：钢琴、提琴、弦乐、吉他、长笛、口琴、古筝
- 冥想：呼吸觉察、身体扫描、正念、睡眠、慈心
- 故事/播客：人文、趣味杂谈、故事奇谈、情感陪伴、知识助眠

### 2. 生成提示词

已为环境音和轻音乐准备可直接复制到生成工具中的英文提示词，并配套中文说明。

轻音乐部分包含统一负向提示词，用于避免：

- 人声 / 歌词
- 鼓点 / 强节奏
- 戏剧性高潮
- 快速旋律
- 恐怖、紧张、悬疑氛围

### 3. TTS 文案

已准备：

- 5 条冥想短版 TTS 种子脚本
- 5 条故事/播客大模型扩写提示词

冥想脚本包含 `<#x#>` 停顿标记，便于后续接 MiniMax 或其他 TTS provider。

### 4. 人工筛选标准

已定义人工听感评分维度：

- 自然度
- 睡前适配
- 稳定性
- 可循环性
- 商用风险

保留标准：

- 平均分 >= 4
- 睡前适配 >= 4
- 稳定性 >= 4
- 无明显版权疑虑

### 5. 候选记录模板

已定义人工试生成后需要记录的字段：

```text
title:
category:
subtype:
tool:
model:
prompt:
duration:
file_path:
license_plan:
quality_score:
notes:
```

## 后续建议

优先测试顺序：

1. 轻音乐：钢琴、古筝、弦乐
2. 环境音：雨声、风扇、海浪
3. TTS：呼吸觉察冥想、情感陪伴

收到人工生成样本和质量反馈后，再进入下一轮工程任务：

1. 确定最终音频来源和商用授权记录方式
2. 替换 `floppy_backend/catalog.py` 中的 placeholder 条目
3. 增加真实音频导入脚本
4. 写入 `storage/pregen/...`
5. 入库 `audio_assets`，让推荐和缓存可以直接命中
